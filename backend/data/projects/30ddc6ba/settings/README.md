# 世界设定目录

这里存放小说的世界观设定文件。

## 推荐结构

```
settings/
├── 世界观.md      # 整体世界观
├── 地理.md        # 地理设定
├── 历史.md        # 历史背景
├── 势力.md        # 势力/组织
├── 规则.md        # 世界规则（魔法/科技等）
├── 时间线.md      # 故事时间线
└── 术语表.md      # 专有名词
```

## 世界观模板

```markdown
# 世界观

## 概述

<!-- 世界观的简要描述 -->

## 核心规则

<!-- 这个世界的基本规则 -->
<!-- 如：魔法系统、科技水平、社会结构等 -->

## 地理

<!-- 世界地图、重要地点 -->

## 历史

<!-- 重要历史事件 -->

## 势力

<!-- 主要势力/组织 -->

## 限制与禁忌

<!-- 世界中的禁忌或限制 -->
<!-- 用于保持故事的一致性 -->
```
