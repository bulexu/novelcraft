# 张爱玲风格指南

张爱玲小说的语言风格特征，适用于细腻心理、都市情感题材。

## 核心特征

### 词汇特征

| 类别 | 特点 | 示例 |
|------|------|------|
| 物质词 | 关注衣物、首饰、家居 | "旗袍"、"翡翠镯子"、"丝绒" |
| 感官词 | 触觉、嗅觉为主 | "凉"、"腻"、"香" |
| 颜色词 | 具体、有质感 | "藕荷色"、"蟹青"、"墨绿" |
| 时间词 | 精确到刻度 | "下午三点钟"、"黄昏时分" |

### 句式特征

**长句缠绕**：
> 她穿着那件藕荷色的旗袍，旗袍上绣着几枝淡雅的兰花，她那翡翠镯子在灯光下发出幽幽的光，她的手指瘦长，指甲涂着淡粉色的指甲油，整只手像是透明的。

**对比反转**：
> 她爱他，却又不肯承认爱他，爱他便成了恨他，恨他反倒成了爱他，她自己也糊涂了。

**比喻从日常生活取材**：
> 她的笑容像是一个疲惫的旅人终于找到了落脚的地方。

### 叙事视角

- **第三人称有限视角**：贴近人物内心
- **冷眼旁观**：叙述者与人物保持距离
- **微物主义**：通过细节暗示整体

## 人物描写模板

### 外貌描写（物质化）

> 她穿一件墨绿的丝绒旗袍，领口绣着金线，耳上坠着翡翠耳环。她的脸是椭圆形的，皮肤白皙，眉毛描得很细。

**要点**：
- 先写衣物（材质、颜色）
- 再写首饰
- 最后写脸部（不夸张）

### 气质描写（含贬抑）

> 她生得不算美，但有股子精明劲儿，眉眼间总带着几分算计。

**要点**：
- 不直接说美或不美
- 用性格词暗示气质
- 带一点讽刺或冷淡

### 年龄描写

> 她三十出头，脸上的细纹已经遮不住了，但保养得当，还算体面。

**要点**：
- 不回避衰老
- 用"遮不住"、"体面"等词

## 对话特征

### 语言风格

| 人物类型 | 语言特点 |
|---------|----------|
| 女性 | 话里有话，表面客气暗含锋芒 |
| 男性 | 敷衍、推脱、不负责任 |
| 长辈 | 传统观念，强势 |
| 晚辈 | 妥协、无奈、隐忍 |

### 对话示例

**暗含锋芒**：
> "你这件事办得倒是体面，我怎么就没想到呢。"她笑了笑，笑意却不达眼底。

**敷衍推脱**：
> "你别说这些了，我都明白。明天再说吧，明天再说。"

**妥协无奈**：
> "好罢，就依你。反正我也没什么主意。"

## 场景描写

### 居家场景

> 房间里摆着一张红木梳妆台，台上放着一面椭圆镜子、几瓶香水、一个翡翠粉盒。窗帘是淡紫色的丝绒，拉得严严实实。

**要点**：
- 物品列举（材质必写）
- 光线描写（灯光、日光）
- 氛围封闭感

### 都市场景

> 饭店的大堂里灯火通明，人声嘈杂。她坐在角落的位置，看着那穿旗袍的女人们走来走去。

**要点**：
- 灯光、噪音
- 人物群像（不具体）
- 主角位置的边缘感

### 情感场景

> 她坐在窗边，看着外面的雨。雨水打在玻璃上，模糊了整个世界。

**要点**：
- 环境暗示心境
- 不直接写悲伤

## 情感表达

### 爱情表达（悲观）

> 她知道他不爱她，但她还是要见他。见了他，她便安心了，安心之后便是新的不安。

**要点**：
- 承认不爱
- 循环式心理（安心→不安）
- 不求解决

### 失望表达

> 她原以为他会来，但他没来。她便也不等了，自己坐到梳妆台前，描了描眉毛。

**要点**：
- 不写"失望"二字
- 用行动代替心理描写

### 妥协表达

> 她想了一想，便也罢了。反正这辈子就是这样了。

**要点**：
- "罢了"、"反正"是关键词
- 顺从命运的姿态

## 常见错误

| 错误 | 正确 |
|------|------|
| 现代流行词 | 保持民国时代感词汇 |
| 直抒胸臆式心理描写 | 通过动作、细节暗示 |
| 过于正面的描写 | 保持一点冷淡、讽刺 |
| 大段落情感爆发 | 含蓄、点到为止 |
| 完美的人物 | 每个人都有缺陷 |
