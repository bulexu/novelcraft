## 设计系统 CSS Tokens

### 颜色系统

```css
:root {
  /* Primary - 主色调 */
  --primary: #4648d4;
  --primary-container: #6063ee;
  --primary-fixed: #e1e0ff;
  --primary-fixed-dim: #c0c1ff;
  --on-primary: #ffffff;
  --on-primary-container: #fffbff;

  /* Secondary - 次色调 */
  --secondary: #575992;
  --secondary-container: #bdbefe;
  --on-secondary: #ffffff;
  --on-secondary-container: #494b83;

  /* Tertiary - 强调色 (用于警告、提示) */
  --tertiary: #904900;
  --tertiary-container: #b55d00;
  --on-tertiary: #ffffff;
  --on-tertiary-container: #fffbff;

  /* Error - 错误色 */
  --error: #ba1a1a;
  --error-container: #ffdad6;
  --on-error: #ffffff;
  --on-error-container: #93000a;

  /* Surfaces - 表面色 */
  --background: #fcf8ff;           /* 页面背景 */
  --surface: #fcf8ff;              /* 基础表面 */
  --surface-container-lowest: #ffffff;   /* 最高对比度区域 (稿件区) */
  --surface-container-low: #f5f2fe;      /* 侧边栏 */
  --surface-container: #efecf8;          /* 次级面板 */
  --surface-container-high: #e9e6f3;     /* AI 面板 */
  --surface-container-highest: #e4e1ed;  /* 工具栏/浮动元素 */
  --surface-variant: #e4e1ed;
  --surface-dim: #dbd8e4;
  --surface-bright: #fcf8ff;

  /* Text - 文本色 */
  --on-background: #1b1b23;
  --on-surface: #1b1b23;           /* 正文文本 */
  --on-surface-variant: #464554;   /* 次级文本 (标签、元数据) */

  /* Outlines - 边框色 */
  --outline: #767586;
  --outline-variant: #c7c4d7;

  /* Inverse - 反色模式 (沉浸写作) */
  --inverse-surface: #303038;
  --inverse-on-surface: #f2effb;
  --inverse-primary: #c0c1ff;
}
```

### Typography 字体系统

```css
:root {
  /* 字体家族 */
  --font-headline: 'Manrope', sans-serif;   /* 标题 */
  --font-body: 'Inter', sans-serif;         /* 正文 (英文) */
  --font-chinese: 'PingFang SC', 'Microsoft YaHei', sans-serif; /* 中文 */

  /* 字号 (基于 2x spacing scale) */
  --display-md: 2.75rem;   /* 章节标题 */
  --headline-lg: 1.5rem;   /* 大标题 */
  --headline-md: 1.25rem;  /* 中标题 */
  --body-lg: 1rem;         /* 正文 */
  --body-md: 0.875rem;     /* 小正文 */
  --label-md: 0.75rem;     /* 标签、元数据 */

  /* 行高 */
  --line-height-display: 1.2;
  --line-height-headline: 1.3;
  --line-height-body: 1.6;
  --line-height-chinese: 1.8;  /* 中文需要更大行高 */
}
```

## 关键组件模式

### 1. Glassmorphism 玻璃态面板

```css
.glass-panel {
  background: rgba(245, 242, 254, 0.8);  /* surface-container-low 80% opacity */
  backdrop-filter: blur(24px);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-ambient);
  /* 无边框！ */
}
```

### 2. Gradient Button 渐变按钮

```css
.btn-primary {
  background: linear-gradient(135deg, var(--primary), var(--primary-container));
  color: var(--on-primary);
  border-radius: var(--radius-md);
  border: none;
  padding: var(--spacing-3) var(--spacing-5);
  font-weight: 500;
}

.btn-primary:hover {
  background: linear-gradient(135deg, var(--primary-container), var(--primary));
}
```

### 3. Ghost Border 幽灵边框

```css
/* 仅在元素可能消失时使用 */
.ghost-border {
  border: 1px solid rgba(199, 196, 215, 0.2);  /* outline-variant 20% opacity */
}
```

### 4. Focus Bead 聚焦点

```css
.focus-bead {
  width: 6px;
  height: 6px;
  background: var(--primary);
  border-radius: var(--radius-full);
  /* 放在当前分析段落旁边 */
}
```

### 5. Surface Hierarchy 表面层次

```css
/* 三栏布局 */
.sidebar {
  background: var(--surface-container-low);  /* #f5f2fe */
  width: 240px;
}

.main-workspace {
  background: var(--surface);  /* #fcf8ff */

  /* 无边框 - 通过色差区分 */

}
```

## 设计原则

### The "No-Line" Rule 无边框规则

**禁止使用 1px solid borders 进行分隔！**

替代方案：
1. **色差分隔**: 使用不同 surface 背景色
2. **间距分隔**: 使用 whitespace
3. **Ghost Border**: 仅在必要时使用 20% opacity 边框

### Tonal Architecture 色调架构

通过背景色变化定义空间：
- 每增加一层 surface-container，背景色加深 6-8 个 hex 单位
- 层级越深，颜色越深

### Breathing Room 呼吸空间

使用 aggressive spacing scale：
- 列表项间距: var(--spacing-4) (16px)
- 卡片内边距: var(--spacing-5) (24px)
- 模块间距: var(--spacing-6) (32px)
