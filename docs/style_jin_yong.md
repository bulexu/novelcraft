# 金庸风格指南

金庸小说的语言风格特征，适用于武侠、历史题材创作。

## 核心特征

### 词汇特征

| 类别 | 特点 | 示例 |
|------|------|------|
| 古典词 | 半文半白，典雅但不晦涩 | "须得"、"便是"、"那厮" |
| 武林术语 | 固定术语体系 | "内力"、"招式"、"轻功"、"穴道" |
| 成语化表达 | 大量四字短语 | "剑气纵横"、"身形如电" |
| 方言融入 | 地域特色词汇 | 北方话的粗豪、江南话的细腻 |

### 句式特征

**对仗工整**：
> 他左手划圈，右手出掌，一招"见龙在田"，啪的一声，击在敌手胸口。

**诗词化嵌入**：
> 那少女身穿淡绿衣衫，犹如出水芙蓉，清丽脱俗。

**动作分解描写**：
> 郭靖这一招"亢龙有悔"打出，劲力刚猛，但留有余地。对手若是抵挡不住，他便收劲；若是对方反击，他便顺势化招。

### 叙事节奏

- **大开大合**：大战前必有铺垫，战后必有余波
- **章回体结构**：每章有悬念结尾
- **穿插说书腔**："话说"、"且说"

## 人物描写模板

### 外貌描写

> 身长八尺，面如冠玉，目若朗星，眉似刀裁。

**要点**：
- 用古典比喻（冠玉、朗星）
- 四字短语组合
- 不写现代感的形容词

### 气质描写

> 他虽是少年，但举止沉稳，眉宇间隐隐有股英气。

**要点**：
- 用"隐隐"、"透出"等含蓄词
- 气质与外貌分开写

### 武功描写

> 他这一招使得行云流水，看似平平无奇，实则暗藏杀机。

**要点**：
- 招式名称必有
- 动作要有评价（行云流水）
- 暗含玄机

## 对话特征

### 人物语言风格区分

| 人物类型 | 语言特点 |
|---------|----------|
| 老者 | 缓慢、说理、带沧桑感 |
| 少侠 | 直接、热血、略带稚气 |
| 女侠 | 温婉或凌厉，依性格定 |
| 反派 | 狂妄或阴冷，话里有话 |

### 对话示例

**武林切磋**：
> "阁下武功精湛，在下佩服。但这招式之中，似乎还有些破绽。"
> "请教。"
> "在下不才，愿与阁下切磋几招，请！"

**恩怨对话**：
> "当年之事，你心中想必清楚。今日相见，不必多言，动手吧。"

## 场景描写

### 山水场景

> 那山峰巍峨耸立，云雾缭绕。山腰处一道瀑布倾泻而下，水声轰鸣，震耳欲聋。

**要点**：
- 先写整体轮廓
- 再写局部细节
- 加入听觉、动态

### 武林场景

> 酒楼之上，群雄聚首。众人各占一方，桌椅狼藉。那为首之人沉声道："今日之事，各位心中清楚。"

**要点**：
- 空间布局交代清楚
- 氛围紧张感
- 关键人物出场

### 夜景

> 月色如水，洒落庭院。树影婆娑，夜风微凉。

**要点**：
- 光线描写先行
- 动静结合

## 情感表达

### 含蓄表达

> 他心中一凛，面上却不动声色。

> 她低下头，心中思绪万千，却说不出话来。

### 决绝表达

> 他咬紧牙关，心想：纵是粉身碎骨，也要护她周全。

## 常见错误

| 错误 | 正确 |
|------|------|
| 现代词汇（"感觉"、"心理"） | 古典词汇（"心中"、"念头"） |
| 西式句式（"他感到一种..."） | 中式句式（"他心头涌起..."） |
| 过多形容词堆叠 | 四字短语组合 |
| 心理分析式描写 | 动作暗示心理 |
