第三章 暗流涌动

雨丝细密地斜织在窗棂上，发出若有若无的沙沙声。在这个静谧得近乎压抑的午后，沈修远独自坐在红木书桌前，手中的狼毫笔尖悬在宣纸上方，迟迟没有落下。浓稠的墨汁在笔尖汇聚，如同一颗黑色的泪珠，映照着阁楼内昏黄的灯火。

楼下的街道传来一阵急促的马蹄声，瞬间打破了雨中的宁静。他眉头微蹙，放下了笔。这种时候，不该有来客。更何况，这马蹄声的节奏如此紊乱，显然并非寻常的出巡。

密函的内容在他脑海中一遍又一遍地回放：“京城有变，速离。”短短六个字，却字字千钧。沈修远深吸一口气，空气中弥漫着陈年墨香与湿冷潮气的混合味道。他知道，这看似平静的南浔古镇，终究也是待不下去了。

他站起身，走到窗边。街角的石桥旁，一个披着斗笠的瘦削身影正冒雨穿过小巷。那人步履匆忙，却在转角处不经意地回了一下头。沈修远的心跳猛然漏了一拍——那是青衣社的标记。

“暗流已经汇聚成川，而我，正立于洪峰之巅。”

他正准备熄灭灯火，门外却响起了三长两短的扣门声。那是他们约定好的信号，却比预定时间早了整整两个时辰。沈修远的右手缓缓滑向袖底，那里藏着一把寒芒内敛的短匕……
