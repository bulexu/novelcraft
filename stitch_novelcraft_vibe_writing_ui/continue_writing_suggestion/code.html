雨夜的访客

窗外的雨声越来越大，密集的雨滴敲打着破旧的瓦片，发出沉闷的响声。陈旧的案几上，那盏油灯已经快要燃尽了，火苗在微弱的穿堂风中摇晃，随时都可能熄灭。

苏宁揉了揉发涩的双眼，长叹一口气。这些卷宗里记载的内容，远比他想象中要复杂得多。本以为只是一桩普通的失踪案，却没曾想牵扯到了十年前那场不为人知的宫廷秘辛。

他站起身，走到窗前，推开了一条缝隙。潮湿的冷风瞬间灌了进来，让他原本有些混沌的大脑清醒了几分。
