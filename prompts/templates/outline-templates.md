## 五、大纲自检清单

```markdown
# 大纲自检

## 弧级检查

- [ ] 每一弧的高潮有没有至少两层以上铺垫
- [ ] 弧末的变化是不是来自角色选择，而不只是事件发生
- [ ] 每一阶段结束后，关系/信息/情绪/局势里有没有至少一项真正变化
- [ ] 下一弧接手时，最不能丢的东西是什么

## 角色检查

- [ ] 每个主要角色在大纲中有自己的目标
- [ ] 每个主要角色有至少一个只属于他/她的选择时刻
- [ ] 角色不能只在"关系轴"里作为主角的附属出现

## 节奏检查

- [ ] 高潮分布是否合理
- [ ] 桥梁章是否足够
- [ ] 有没有连续太长的高强度段落
- [ ] 有没有连续太长的平淡段落

## 线索检查

- [ ] 主要伏笔是否有回收计划
- [ ] 回收时机是否合理
- [ ] 是否有过期未收的线索
```
